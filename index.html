Smart Sensor Technology

IoT-powered smart bins with real-time monitoring capabilities

Ultrasonic Sensors(HC-SR04)

Measure distance to existing objects to determine waste fill level inside bins

Real-Time Level Detection

Weight Sensors

Load sensors quantify actual mass of collected waste for prediction and optimization using the load cell strain gauge

Mass Quantification
